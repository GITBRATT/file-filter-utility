## File Filter Utility (Java: OpenJDK 25.0.1)
Консольная утилита для фильтрации содержимого текстовых файлов.  
Программа читает строки из указанных пользователем файлов, определяет тип данных
(целое число, вещественное число или строка), распределяет данные по отдельным
выходным файлам и выводит статистику по каждому типу.

## Инструкция
- После приветствия программа предлагает ввести имена файлов для сканирования через запятую.
- Далее предлагает ввести настроечные команды:
  Ключ	Описание
  -"-q"	начать фильтрацию
  -"-e"	заново выбрать файлы
  -"-o" <путь>	задать каталог вывода
  -"-p" <префикс>	задать префикс имен файлов
  -"-a"	режим добавления (append)
  -"-s"	краткая статистика
  -"-f"	полная статистика
- По окончанию конфигурации вводим команду "-q", далее утилита читает файлы и автоматически определяет тип строки:
  - целые числа (long)
  - вещественные числа (double, поддерживается "," и ".")
  - строки
- записывает результаты в отдельные файлы:
  - "integers.txt"
  - "floats.txt"
  - "strings.txt"
- собирает статистику:
  - для чисел: количество, минимум, максимум, сумма, среднее
  - для строк: количество, минимальная и максимальная длина
- поддерживает режим добавления в файл (append)
- поддерживает префикс имен выходных файлов
- позволяет задать каталог вывода

## Особенности реализации
- Чтение файлов построчно через BufferedReader
- Статистика собирается отдельным классом "Stats"
- Выполняется проверка существования входных файлов
- Выходные директории создаются автоматически при необходимости
